For python users:
1. Execute from shell like this: $ ./tableFormat.py data.txt
---------------------------------------
For php users:
1. Find out path to where your php installation is type in shell: $ whereis php (in most cases: '/usr/bin/php' )
2. Append #!/path/to/php (in most cases: #!/usr/bin/php) to beginning of the script
3. Make the php script executable - $ chmod -x scriptName.php (or $ chmod 777 scriptName.php)
4. Execute like this: $ ./tableFormat.php data.txt
